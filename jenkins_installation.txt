1. installing jenkins on EC2 instance
2. Install java
apt-get install openjdk-7-jre
3. Install Jenkins
wget -q -O - http://pkg.jenkins-ci.org/debian/jenkins-ci.org.key | sudo apt-key add -
sudo sh -c 'echo deb http://pkg.jenkins-ci.org/debian binary/ > /etc/apt/sources.list.d/jenkins.list'
sudo apt-get update
sudo apt-get install jenkins

4.Installing Apache
sudo apt-get install apache2
sudo a2enmod proxy
sudo a2enmod proxy_http

5. Change jenkins.conf file to proxy from 80 to 8080 port
/etc/apache2/sites-available/jenkins.conf

<VirtualHost *:80>
    ServerName HOSTNAME
    ProxyRequests Off
    <Proxy *>
        Order deny,allow
        Allow from all
    </Proxy>
    ProxyPreserveHost on
    ProxyPass / http://localhost:8080/
</VirtualHost>

6. Enable Jenkins.config
      sudo a2ensite jenkins
      sudo service apache2 reload

7.Installing Java / Maven / Git
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer maven git-core
